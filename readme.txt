== Description ==
This plugin is exclusively for members of the Influence Central community. For more information, please visit www.influence-central.com

== Installation ==
To install the plugin, simply upload the zip file to the plugins directory or use the automated installer. Once installed, click the \"settings\" link to enter your Influence Central ID and click \"Save\".

Additional information can be found by logging in to the member portal at http://www.influence-central.com/
